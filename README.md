# Welcome to Atlas

Atlas is a cross-platform* 2D game engine, built to support my personal game development. Many architectural decisions and development tools were based on @TheCherno's *Hazel* game engine tutorial series, but I've made significant changes to streamline for my own purposes. 

*It will be.

___

### Dependencies
- CMake
- Glad
- GLFW
- ImGui
- Metal-Cpp
- Spdlog
- Stb_image
